# SEO Audit: learninfrance.com
**Date:** April 7, 2026
**Overall SEO Health Score: 32/100**

## Executive Summary

The site has serious SEO issues that prevent search engines from properly indexing and ranking it. The homepage renders correctly with good content, but fundamental technical SEO infrastructure is missing.

### Top 5 Critical Issues
1. **No robots.txt** — returns 404
2. **No sitemap.xml** — returns 404
3. **All pages share the same title/description/canonical** — search engines can't distinguish pages
4. **No structured data (Schema.org)** — zero JSON-LD blocks
5. **No security headers** — missing HSTS, CSP, X-Frame-Options

### Top 5 Quick Wins
1. Add robots.txt and sitemap.xml (Next.js built-in support)
2. Add unique title + description + canonical per page
3. Add Organization + WebSite + EducationalOrganization schema
4. Add security headers via Next.js config
5. Add unique page-level metadata for Why France, Your Journey, Advisor pages

## Priority Action Plan

### Critical (Fix Immediately)

**1. Add robots.txt**
Create `apps/web/public/robots.txt`:
```
User-agent: *
Allow: /
Disallow: /portal/
Disallow: /dashboard/
Disallow: /auth/
Sitemap: https://learninfrance.com/sitemap.xml
```

**2. Add sitemap.xml**
Use Next.js built-in sitemap generation — create `apps/web/src/app/sitemap.ts`:
```ts
export default function sitemap() {
  return [
    { url: 'https://learninfrance.com', lastModified: new Date(), priority: 1.0 },
    { url: 'https://learninfrance.com/why-france', lastModified: new Date(), priority: 0.8 },
    { url: 'https://learninfrance.com/your-journey', lastModified: new Date(), priority: 0.8 },
    { url: 'https://learninfrance.com/advisor', lastModified: new Date(), priority: 0.9 },
    { url: 'https://learninfrance.com/partners/burgundy-school-of-business', lastModified: new Date(), priority: 0.7 },
  ]
}
```

**3. Add unique metadata per page**
Each page in `(public)/` needs its own `export const metadata` with unique title, description, and canonical.
